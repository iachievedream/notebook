開啟Terminal(終端機):按 ctrl+alt+T

install apache2

~~~
sudo apt-get install apache2
~~~

後面輸入密碼




ls /etc/apache2

sudo cp apache2.conf apache2org.conf

開啟vi

~~~
vi
~~~



開啟vi的test.txt檔案:vi test.txt
編輯:a與i鍵

退出esc
儲存:w
離開
:q

~~~
h: 向左一個字元
l: 向右一個字元
k: 向上一行
j: 向下一行
~~~
